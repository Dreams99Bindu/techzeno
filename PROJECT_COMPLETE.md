## 🎨 Quick Customization Guide

### Change Colors
Edit `src/styles/variables.scss`:
```scss
$primary-blue: #0066ff;      // Your brand color
$secondary-blue: #00d4ff;    // Secondary color
```

### Update Content
Edit `src/app/services/mock-data/mock-data.ts`:
- Update services, projects, pricing, team info

### Add New Pages
1. Create: `ng g c pages/my-page --standalone`
2. Add route in `src/app/app.routes.ts`
3. Add nav link in navbar
